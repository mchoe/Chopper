A vagrant file and bootstrap script that installs everything necessary for a Flask/Postgres/Alchemy setup

1. `git clone git@github.com:mchoe/PythonWeb.git`
2. `cd PythonWeb`
3. `vagrant up`
4. `vagrant ssh`
5. `cd /vagrant`
6. `python bootstrap.py`
7. Go!
